# Support

Tribunus is a **pre-alpha** macOS desktop cockpit. As such, response times and issue resolution are not guaranteed. Community and commercial support options are outlined below.

## Bug Reports

Report bugs via [GitHub Issues](https://github.com/TribunusHQ/Tribunus/issues) using the **bug report** template. Include reproduction steps, expected vs actual behavior, and your environment details.

## Questions

Ask questions through [GitHub Discussions](https://github.com/TribunusHQ/Tribunus/discussions) or open an [Issue](https://github.com/TribunusHQ/Tribunus/issues) using the **question** template.

## Security Issues

Report security vulnerabilities privately via [SECURITY.md](SECURITY.md) or the **GitHub Security Advisory** tab on the repository.

## Commercial Support

For commercial support, contact **hello@tribunus.dev**.
