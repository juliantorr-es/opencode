Search GitHub pull requests by title and description. Uses GITHUB_TOKEN environment variable for authentication.

PARAMETERS:
- `owner` (required): Repository owner (user or organization)
- `repo` (required): Repository name
- `query` (required): Search query for PR titles and descriptions
- `state` (optional): PR state filter: "open", "closed", or "all" (default "open")
- `limit` (optional): Maximum number of results (default 10)
- `offset` (optional): Number of results to skip for pagination (default 0)

OUTPUT: Structured PR list with title, URL, state, and labels for each match.
