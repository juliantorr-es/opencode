Record a prepublication review verdict of 'admitted' for a candidate boundary.
Writes a single JSON record under docs/json/opencode/approvals/prepublication_admitted.v1.json.
Each record captures schema_version, candidate_packet_digest, candidate_boundary_identifier,
parsed reviewer_set, review_round, status, notes, session_id, and an ISO timestamp.

Output format:
  {"status":"admitted","candidate_boundary_identifier":"...","review_round":...}
