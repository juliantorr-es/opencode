Welcome to the documentation bootstrap for **Tribunus**.

## Getting Started
Tribunus is an open-source AI coding agent designed to help you write and run code with any AI model.

## Installation & Launch Notes
To install and launch the desktop application, download the appropriate release installer for your operating system:
- **macOS**: `tribunus-desktop-darwin-arm64.dmg` or `tribunus-desktop-darwin-x64.dmg`
- **Windows**: `tribunus-desktop-win-x64.exe`
- **Linux**: `tribunus-desktop-linux-x64.AppImage`

Once installed, launch the application from your Applications menu or run the launcher utility.

## Compatibility Note
To ensure backwards compatibility and prevent breaking transitions:
- Configuration filenames remain `opencode.json` and `.opencode/`.
- Environment variables (`OPENCODE_DB`, `OPENCODE_CHANNEL`, etc.) and system headers (`x-opencode-directory`) remain unchanged.
- The command-line utility command interface remains `opencode`.
